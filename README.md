## Quick Start

### Prerequisites
- Docker & Docker Compose

### Run with Docker
`git clone https://github.com/YOUR_USERNAME/Cafe-Manager.git`

`cd gic-platform`

`docker-compose up --build`

Then visit:
- **Frontend:** http://localhost
- **Backend API:** http://localhost:8000
- **API Docs:** http://localhost:8000/docs


## Stop Application

`docker-compose down`
